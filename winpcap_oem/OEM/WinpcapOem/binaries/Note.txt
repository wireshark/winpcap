This folder will contain the binaries for the various OSes that will be
embedded in the resources:

Packet2k.dll: Windows 2k/XP/2003 packet.dll
Packetnt.dll: Windows NT packet.dll
PacketVista.dll: Windows Vista packet.dll
npf2k.sys: Windows 2k/XP/2003 driver
npfnt.sys: Windows NT driver
npfx64.sys: Windows AMD64 driver

These files are copied automatically by the build process system in makeall
